推荐工作流

  1. 把原始 PDF/源码/草稿放入 papers_sources
  2. 在 notes 产出单篇阅读笔记（建议用模板）
  3. 将单篇结论沉淀到 index
  4. 抽取可复用方法、术语、公式到 index
  5. 在 index 做跨论文对比和总结
