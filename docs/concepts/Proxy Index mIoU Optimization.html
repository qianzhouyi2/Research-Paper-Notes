Proxy Index mIoU Optimization

概念

在无法访问梯度和置信度时，用可观测指标驱动搜索。DLA 选择 mIoU 作为代理指标，每次只接受能进一步降低 mIoU 的扰动更新。

为什么是 mIoU

  • 比 PAcc 更能反映分割的整体结构质量。
  • 与数据集评估口径一致，便于跨样本比较。

影响

  • 将“更新是否有效”变成可比较的单调准则。
  • 降低随机搜索中的无效查询比例。^[inferred]

风险

  • 如果目标是特定类别破坏，单一 mIoU 可能过粗。
  • 代理指标与真实任务目标不一致时，会产生优化偏差。^[inferred]
