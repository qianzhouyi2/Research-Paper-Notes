# Cityscapes Dataset

## 联网补充

- Cityscapes 官方页面把它定义为城市场景语义理解数据集，包含 5000 张高质量精标注图像、20000 张粗标注图像，覆盖 50 个城市。
- 在本库里，Cityscapes 是语义分割黑盒攻击的主评测场，重点不是类别本身，而是查询预算下 mIoU 能被压低多少。
